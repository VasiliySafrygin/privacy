# Политика конфиденциальности мобильного приложения The Pusher

Дата вступления в силу: 13 апреля 2026 г.

Эта Политика конфиденциальности описывает, какие данные могут обрабатываться при использовании мобильного приложения **The Pusher** (далее — «Приложение»), и как они используются.

## 1. О приложении

`The Pusher` (Толкач) — мобильная игра-головоломка.  
Приложение работает преимущественно офлайн и сохраняет игровой прогресс локально на устройстве.

## 2. Какие данные мы обрабатываем

### 2.1. Данные, которые хранятся локально на устройстве

Приложение может сохранять:
- игровой прогресс (уровень, очки, жизни);
- настройки приложения (например, включение/выключение музыки и звуков);
- технический флаг покупки «отключение рекламы» (если применимо).

Эти данные хранятся локально в памяти устройства и используются только для работы Приложения.

### 2.2. Рекламные данные (сторонний SDK)

В Приложении используется рекламный SDK **Yandex Mobile Ads** для показа рекламы (баннеры, rewarded-реклама).  
При работе рекламы соответствующий SDK может автоматически обрабатывать технические данные в соответствии со своей политикой, в том числе:
- рекламные идентификаторы устройства;
- IP-адрес;
- технические параметры устройства и приложения;
- данные о показах/кликах/эффективности рекламы.

Подробности обработки таких данных определяются политиками Яндекса:
- [Политика конфиденциальности Яндекса](https://yandex.ru/legal/confidential/)
- [Документация Yandex Mobile Ads](https://yandex.ru/dev/mobile-ads/)

## 3. Для чего используются данные

Данные используются для:
- сохранения игрового прогресса и настроек;
- показа и измерения эффективности рекламы;
- обеспечения стабильной работы Приложения.

## 4. Передача данных третьим лицам

Разработчик Приложения не продает персональные данные пользователей.  
Передача технических данных может происходить только через интегрированные сторонние сервисы (например, Yandex Mobile Ads) в пределах, необходимых для их работы.

## 5. Права пользователя

Вы можете:
- прекратить использование Приложения в любой момент;
- удалить локальные данные Приложения через настройки устройства (очистка данных/удаление Приложения);
- отключить персонализацию рекламы средствами ОС (если такая опция поддерживается устройством).

## 6. Безопасность

Мы принимаем разумные меры для защиты данных в рамках обычной работы Приложения.  
Однако ни один способ хранения или передачи данных не гарантирует абсолютную безопасность.

## 7. Дети

Приложение не предназначено для целенаправленного сбора персональных данных детей.  
Если вы считаете, что данные ребенка были обработаны некорректно, свяжитесь с нами.

## 8. Изменения политики

Мы можем обновлять эту Политику конфиденциальности.  
Актуальная версия публикуется по адресу, указанному в карточке Приложения (Google Play / сайт разработчика).
